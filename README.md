##页面分类

* 任务页
	* 时间统计：项目/任务的起止时间？
	* 项目做完一个划掉一个
	* 任务大小/难易程度
	* 标签（地点、分类、自定义）√
	* 紧急程度
* 便笺页（备忘、记录突发事件）
	* 日期、内容
	* 快速转换成任务
* 成就页
	* 种树：浇水、成长值
* 统计页
	* 当天/周/月的完成情况等。（echart）
* 打卡页
	* 习惯养成
* 垃圾桶页
	* 恢复已删除的信息
* 个人中心页
	* 任务标签管理
	* 个人信息修改
* 注册页√
	* 对邮箱进行查重校验
	* 对邮箱格式进行校验√
	* 密码强弱校验√
	* 重复密码校验√
	* 判空验证√
	* 验证码
